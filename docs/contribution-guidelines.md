# دليل المساهمة في المشروع

## قبل المساهمة
يجب قراءة ملف `ETHICS_POLICY.md` بالكامل والموافقة عليه. لن تُقبل أي
مساهمة تخالف سياسة الاستخدام الأخلاقي.

## أنواع المساهمات المقبولة
- توثيق سيناريو محاكاة جديد باستخدام القالب الموحد.
- تحسين سكربتات التحليل في `pattern-extraction/`.
- إضافة قاعدة كشف جديدة مدعومة بنمط مُحلَّل ومُوثَّق.
- تحسين الوثائق أو تصحيح أخطاء.

## خطوات تقديم مساهمة
1. Fork للمستودع.
2. إنشاء فرع جديد بوصف واضح: `feature/scenario-t1078-lateral-movement`
3. تنفيذ التغييرات مع الالتزام بهيكل المجلدات الحالي.
4. التأكد من عدم تضمين أي بيانات حقيقية أو حساسة في أي ملف.
5. تقديم Pull Request مع وصف واضح للمساهمة والهدف البحثي منها.

## معايير المراجعة
كل Pull Request يخضع لمراجعة من حيث: الالتزام بسياسة الاستخدام الأخلاقي،
جودة التوثيق العلمي، وعدم تضمين أي كود استغلالي أو أدوات هجومية جاهزة.

## التواصل
لأي استفسار، افتح Issue جديد مع وصف واضح للمشكلة أو الاقتراح.
